# Vertical de Web (Inteligência Viva)

Esta vertical dota o sistema Lessie AI da capacidade de ler o mundo real em tempo real.

## Habilidades Principais

### 1. `web_search`
**Real-time Indexing.**
- **Gatilho**: Quando for necessário encontrar pegadas digitais recentes, notícias de última hora ou validar associações extrínsecas.
- **Função**: Sonda a internet global via APIs de busca ao vivo.

### 2. `web_fetch`
**Deep Scraping.**
- **Gatilho**: Para análise profunda de perfis técnicos (GitHub), sites institucionais ou artigos de blog.
- **Função**: Extrai e analisa o DOM de URLs específicos para capturar nuances técnicas e conquistas.

## Regras de Operação
- Sempre prefira filtrar o ruído do DOM no Agente de Conexão.
- Use `web_fetch` para alimentar a argumentação lógica do Agente de Conexão.
