# ZÉLLA SMARTHTEL — DESIGN SYSTEM CONTEXT (`.impeccable.md`)

Este arquivo define as diretrizes estritas de design, paleta de cores, classes utilitárias e regras contra "AI Slop" para a landing page e painéis do **Seu Zélla**. Deve ser consultado por qualquer agente de IA antes de realizar modificações ou criar novos componentes de frontend.

---

## 🎨 1. Diretrizes de Cores e Paletas
Evite paletas genéricas de IA (como tons degradê de roxo/violeta com ciano sobre fundo preto puro). O Seu Zélla utiliza uma paleta de marca exclusiva:

*   **Fundo (Canvas)**: `#0a0a0a` (Preto profundo com matiz quente) e `#0a0e1a` (Grafite azulado para seções escuras).
*   **Cor Primária**: Azul Royal (`#4169E1`) para links estruturais, botões secundários e anéis de foco.
*   **Cor de Destaque (Accent)**: Teal (`#14b8a6`) e Verde Esmeralda (`#10b981`) para sinalizar sucesso, conversão de reservas e ações de destaque.
*   **Tipografia**:
    *   **Texto Principal**: `#f1f5f9` (Branco suave)
    *   **Texto Secundário**: `#94a3b8` (Cinza médio)
    *   **Texto Muted/Desativado**: `#64748b` (Cinza escuro)

---

## 💎 2. Utilitários CSS Reais (do `globals.css`)
Não invente classes de vidro, desfoque ou degradê que não estejam declaradas. Use as classes utilitárias nativas do projeto:

*   **Efeito de Vidro (Glassmorphism)**:
    *   `.glass-card`: Fundo branco translúcido (`rgba(255, 255, 255, 0.04)`) com desfoque de `20px` e borda suave.
    *   `.glass-card-hover`: Mesma base do glass-card, mas adiciona hover com elevação de borda, opacidade maior e sombra de `32px`.
    *   `.glass-input`: Fundo input e focus translúcido com transição de foco Azul Royal.
*   **Texto em Degradê**:
    *   `.gradient-text-royal`: Degradê de Azul Royal (`#4169E1`) para Teal (`#14b8a6`). Use de forma moderada apenas em headlines ou métricas-chave.
    *   `.gradient-text-gold`: Degradê ouro-slate-ouro (`#f59e0b` para `#f1f5f9` para `#f59e0b`).
*   **Glow e Sombras**:
    *   `.glow-royal`: Sombra difusa em azul royal.
    *   `.glow-teal`: Sombra difusa em teal.
    *   `.glow-success`: Sombra difusa esmeralda.
    *   `.animate-pulse-glow`: Animação infinita de pulsação e glow azul royal.
*   **Padrões de Fundo**:
    *   `.bg-grid-pattern`: Grade quadriculada suave (`60px`).
    *   `.bg-dot-pattern`: Pontilhado decorativo sutil (`24px`).

## 🚫 3. Anti-Patterns a Evitar (Prevenção de "AI Slop")
O Impeccable Audit mapeou as seguintes práticas que sinalizam design de baixa qualidade e devem ser evitadas:

1.  **Uso Indiscriminado de Degradês de Texto**: Não adicione degradês em títulos que não necessitem de destaque central. Se o título for longo, use cor sólida no bloco e reserve o degradê apenas para a palavra-chave.
2.  **Roxo e Violeta com Ciano**: Proibido usar gradientes de fundo roxo-para-azul no estilo padrão de templates web genéricos. A marca baseia-se em Azul Royal e Verde Esmeralda/Teal.
3.  **Aninhamento Excessivo de Cards**: Não crie estruturas de cards dentro de cards com múltiplas bordas brancas destacadas. Prefira espaçamento em branco para separar conteúdo.
4.  **Bordas Claras no Escuro**: Em fundos escuros, use bordas com opacidade máxima de `3%` a `8%` (`border-white/[0.05]`) para que o layout se integre de forma harmônica e pareça premium.

## 🎬 4. Padrões de Movimento e Animação
Todas as seções devem usar a biblioteca Framer Motion seguindo este padrão:

```typescript
import { useRef } from 'react';
import { motion, useInView } from 'framer-motion';

export function Component() {
  const ref = useRef(null);
  const isInView = useInView(ref, { once: true, margin: '-100px' });

  return (
    <motion.div
      ref={ref}
      initial={{ opacity: 0, y: 30 }}
      animate={isInView ? { opacity: 1, y: 0 } : {}}
      transition={{ duration: 0.6 }}
    >
      {/* Conteúdo */}
    </motion.div>
  );
}
```
*   **Gatilho**: Sempre utilize `once: true` no `useInView` para evitar re-executar a animação de entrada enquanto o usuário rola a página de volta para cima.
*   **Suavidade**: Prefira transições limpas (`opacity` e deslocamento em `y`) a rotações ou zooms extravagantes.
