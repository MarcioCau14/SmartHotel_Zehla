## PASSO 4 — GitHub Secrets (Settings > Secrets and variables > Actions):

##   VERCEL_TOKEN          → Token de acesso Vercel

##   VERCEL_ORG_ID         → ID da organização Vercel

##   VERCEL_PROJECT_ID     → ID do projeto Vercel

##   RAILWAY_TOKEN         → Token de acesso Railway

##   RAILWAY_SERVICE_ID    → ID do serviço Railway (realtime-service)

##   CRON_SECRET           → Segredo para proteger endpoints cron
