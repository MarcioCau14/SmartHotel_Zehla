# Design System ZEHLA (Atualizado)

Este documento estabelece as diretrizes visuais unificadas para a Página de Vendas, ZCC Admin e Dashboard do Cliente.

## 🎨 Paleta de Cores

Para garantir uma leitura confortável e uma estética premium, suavizamos o tom de laranja fluorescente original.

### Cores Principais
- **Background (Escuro):** `#0F172A` (Slate 900) - Um tom de grafite azulado profundo, mais suave que o preto puro.
- **Primary / Accent (Laranja Suave):** `#F97316` (Orange 500) - Laranja vibrante porém equilibrado, confortável para os olhos.
- **Secondary:** `#334155` (Slate 700) - Para bordas, divisores e containers secundários.
- **Text Primary:** `#F8FAFC` (Slate 50) - Quase branco para máximo contraste e legibilidade.
- **Text Secondary:** `#94A3B8` (Slate 400) - Cinza suave para informações de apoio.

### Status & Feedback
- **Success:** `#10B981` (Emerald 500)
- **Warning:** `#F59E0B` (Amber 500)
- **Error:** `#EF4444` (Red 500)

## Typography

- **Títulos (Display):** `Outfit` (ou fallback `system-ui`) - Moderna, limpa e com excelente peso visual.
- **Texto Corrido (Body):** `Inter` - Focada em máxima legibilidade em telas.

## 📐 Componentes Visuais (Guardrails)
- **Bordas:** Arredondamento suave (`0.5rem` / `8px`).
- **Cards:** Fundo `#1E293B` (Slate 800) com bordas sutis.
- **Sombra:** Efeitos suaves (Soft Shadows), evitando contrastes agressivos.
