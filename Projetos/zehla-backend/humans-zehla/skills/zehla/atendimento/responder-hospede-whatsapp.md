# Resposta ao Hóspede via WhatsApp

## Quando usar
Quando um hóspede envia mensagem via WhatsApp com perguntas, solicitações ou reclamações.

## Princípios
1. Velocidade: Responder em até 5 minutos
2. Personalização: Usar nome do hóspede e detalhes da reserva
3. Empatia: Demonstrar cuidado genuíno
4. Proatividade: Antecipar necessidades
5. Clareza: Informações completas e organizadas

## Categorias de mensagens

### INFORMAÇÃO BÁSICA
Tópicos: horários, endereço, estacionamento, Wi-Fi, check-in/out
```
Olá [NOME]! 😊

[RESPOSTA À PERGUNTA]

Mais informações úteis:
📍 Endereço: [ENDERECO]
⏰ Check-in: 14h | Check-out: 11h
🅿️ Estacionamento: [INFO]
📶 Wi-Fi: [REDE] | Senha: [SENHA]

Precisa de mais alguma coisa? 👋
```

### RECLAMAÇÃO
```
[NOME], peço sinceras desculpas pelo inconveniente. 😔

Já tomei as seguintes providências:
1️⃣ [AÇÃO IMEDIATA 1]
2️⃣ [AÇÃO IMEDIATA 2]

Para compensar: [COMPENSAÇÃO]

Sua satisfação é nossa prioridade! 🙏
```

### ELOGIO
```
[NOME], que alegria ler isso! 🥰

Ficamos muito felizes que [DETALHE DO ELOGIO].

Compartilhe sua experiência no Google:
🔗 [LINK DO GOOGLE REVIEW]

Desconto especial para próxima visita:
🎁 Código [NOME]10 → 10% de desconto!

Até breve! 🌴
```

## Regras de formatação WhatsApp
- Máximo 1600 caracteres por mensagem
- Emojis moderados (1-2 por bloco)
- Negrito: *texto* | Itálico: _texto_
- Nunca enviar após 21h
- Pedir permissão antes de imagens/promoções
